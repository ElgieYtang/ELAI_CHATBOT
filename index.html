BSIT Final Project

AI-Powered Website Chatbot

About This Project

This website is developed as a final academic project by a 4th year Bachelor of Science in Information Technology (BSIT) student from the University of Cebu – Main Campus.

Project Objective

The objective of this project is to design and deploy an AI-powered chatbot that can be integrated into a website using open-source tools.

Technologies Used

  • HTML, CSS, JavaScript
  • Flowise
  • Ollama (Local LLM)
  • RAG Architecture
  • Cloudflare Tunnel
